# Frontend Palindrome Checker

A simple frontend JavaScript app to check whether a given input is a palindrome or not, based on alphanumeric characters.

### [Live Demo](https://cillb.github.io/Front-End-Palindrome-Checker/)

---

## Getting Started

### 1. Clone the Repo

    git clone https://github.com/cillb/Front-End-Palindrome-Checker.git
    cd Front-End-Palindrome-Checker

### 2. Open in Browser

Open `index.html` directly in your browser.  
Or visit the [Live Demo](https://cillb.github.io/Front-End-Palindrome-Checker/).
